Error
=====

A flexible error result type.

.. admonition:: Info

   To use declarations from this page, include the ``<essence/error.hpp>`` header.

.. cpp:namespace-push:: essence

.. cpp:class:: error

   A flexible error type, great for use with :cpp:expr:`std::expected`.

      Get the error number.

   .. cpp:function:: error& number(i32 n)

   .. cpp:function:: void die() const noexcept

Environment
===========

Environment variable access and manipulation functions and facilities.

.. admonition:: Info

   To use declarations from this page, include the ``<essence/environment.hpp>`` header.

.. cpp:namespace-push:: essence

.. cpp:function:: std::string env::get(std::string_view key, std::string_view fallback = "")

   Get the value of an environment variable by name, with a fallback value
   when the variable is not set.

   Parameters may also be ``const std::string&`` or ``const char*``.

.. cpp:function:: bool env::set(std::string_view key, std::string_view value, bool clobber = false)

   Set the environment variable ``key`` to ``value``. If
   clobber is ``true``, overwrite an existing value.
   
   Parameters may also be ``const std::string&`` or ``const char*``.

.. cpp:function:: bool env::unset(std::string_view key)

   Unset an environment variable by name. Returns true upon success, false otherwise.
   
   Parameter ``key`` may also be ``const std::string&`` or ``const char*``.

.. cpp:function:: std::vector<std::string> env::vec(char* const* envs = environ)

   Construct a ``std::vector`` from the a null-terminated environment array.
   Each member of the result is the entire pair, including the ``=``.

   Uses the value of `environ <https://man7.org/linux/man-pages/man7/environ.7.html>`__ unless explicitly passed a value.

.. cpp:function:: std::unordered_map<std::string, std::string> env::map(char* const* envs = environ)

   Construct a :cpp:expr:`std::unordered_map` from the null-terminated environment array.
   
   Uses the value of `environ <https://man7.org/linux/man-pages/man7/environ.7.html>`__ unless explicitly passed a value.

.. cpp:var:: detail::environment_t environment

   Global proxy object instance for convenient but safe environment access
   and manipulation. Provides null-safe access to the POSIX ``getenv``,
   ``setenv``, ``unsetenv`` family of functions.

   :cpp:expr:`detail::environment_t` is an implementation detail, all interaction
   should be go through this object.

   .. rubric:: Usage

   .. code-block:: cpp
      
      std::string path = environment["PATH"];
      std::string shell = environment.get("SHELL", "/bin/sh");
      
      environment.set("SOME_THING", "foo");
      environment.set("SOME_THING", "bar", /*clobber=*/ true);

      environment.unset("SOME_THING");
